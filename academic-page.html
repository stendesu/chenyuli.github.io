First Side Scroller Project

• Unreal Engine 4 (Blueprints)

This side scroller game is my Very First game project made during college, this was a simple side scroller game where the players must defeat enemies using a single dash mechanic, the player is able to dash horizontally depending on which direction the player is facing and vertically, each dash will do damage to the enemies. A separate boss level was created with fully functional simple melee AI mechanics. Each stage was made with mechanics in thought, with traps and difficult jumps.

All assets are made by Me using Clip Studio and Photoshop
